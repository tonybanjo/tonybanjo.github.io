Week 2: HTML!

After a week of set-up and intro to the course we were finally ready to code! HTML was our first assignment, and while the basics were mostly refresher. I had a lot to learn! HTML5 has evolved quite a bit from when I created my first website.
Our assignments were:

  • create a page with several different types of HTML tags
  • create a page that highlighted different examples of HTML attributes
  • create a page that had nested tags

My 90's Website

All 90's websites started with an animated sitck man doing construction (or at least the way I remember it!). To get started I explored different HTML tags with minimal formatting. It reminded me of my website from the 90s, so I ran with that theme.

My Splash Page / About Me Image

Back in the day, this image would have taken several minutes to load, haha! Some of the first HTML tags I learned back in the 90s were:

<a href="">, <img src="">, <h1>, <p>, <br>, and <center>

Honestly, I don't remember how background colors, font colors, or any styling was accomplished. I do know that my page would have had a ton of random images and gif to make it look interesting!

...and in typical 90's style. The site is just a collection of things I like. This served as a starting point to look back at different HTML tags, but also some inspiration going forward. (Note, I have used a little bootstrap for the final project)
