Week 6: Bootstrap

Week 6 was the Bootstrap assignment. I was excited for this assignment, because I have run into the problem of designing for my laptop screen. Spending hours on a meticulous layout only to find a hot mess when viewed on another device or browser. Bootstrap solves that issue. It has been challenging, yet fun to learn. There are so many pre-defined classes it can be hard to memorize without constantly consulting the bootstrap documentation page. Once I had a few of them down, I noticed how much time I saved not having to create a class for each element. Below are some examples of elements that use Bootstrap.
