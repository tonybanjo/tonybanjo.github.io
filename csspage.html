Week 3: CSS!

Week 3, CSS! This week was a little bit of refresher with the basics, but the animation abilities of CSS was relatively new. I had previously used CSS to animate navigation bars or button. It was usually a struggle to figure out...You can see below that I really enjoyed CSS animation. I decided to up the stakes on my cat page, adding floating and spinning cat heads throughout the page. While this type of layout is not practical, it was a lot of fun to make!

Aside from having a blast animating cat heads, my biggest takeaway from week 3 was CSS specificity. Previously, I had struggled with getting certain styled elements to work. I realize now that I probably had many conflicting lines of code, but did not understand that inline > embedded > id > class > * . As a page or site develops, some css styles go unused, renamed, or are page specific. I am still continuing to improve in keeping code organized and streamlined.
